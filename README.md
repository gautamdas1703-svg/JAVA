# Smart Student Manager System

## Description

This is a Java-based console application to manage student records. It allows users to add, search, update, delete, and store student data using file handling.

## Features

* Add Student
* View All Students
* Search by Roll Number
* Update Marks & Attendance
* Delete Student
* Automatic Grade Calculation
* Save & Load Data from File

## Concepts Used

* Object-Oriented Programming (OOP)
* Inheritance
* ArrayList
* Exception Handling
* File Handling

##  How to Run

```bash
javac *.java
java Main
```

## File Structure

* Main.java
* Person.java
* Student.java
* StudentManager.java
* FileHandler.java
